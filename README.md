## üöÄ Quick Start

### HIKARU AI Platform
```bash
cd hikaru-nextjs
npm install
cp .env.example .env.local
# Add your API keys to .env.local
npm run dev
```

### VS Code Theme
1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Install from VSIX: `PERESELKA/black-sun-0.0.2.vsix`
4. Select theme: File > Preferences > Color Theme > Black Sun
